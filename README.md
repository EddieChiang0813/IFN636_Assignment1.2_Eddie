# Eddie Forms

Eddie Forms is a web-based feedback form system developed for IFN636 Assignment 1.2.  
It allows users to register, log in, create feedback forms, edit existing forms, delete forms, and manage their own form content through a simple interface.

## Features

- User registration and login
- User authentication with JWT
- Create feedback forms
- View created forms
- Edit forms
- Delete forms
- Protected user-specific form access
- Frontend and backend integration
- Backend automated test cases
- GitHub Actions CI/CD workflow setup
- EC2, PM2, and Nginx deployment configuration

## Tech Stack

### Frontend
- React
- Axios
- React Router

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

### DevOps / Deployment
- GitHub
- GitHub Actions
- AWS EC2
- PM2
- Nginx
